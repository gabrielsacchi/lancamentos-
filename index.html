⚙ Configurar planilha

Para que o app envie direto pra sua planilha, é preciso criar um Apps Script no Google Sheets. Siga os passos:

1. Abra sua planilha no Google Sheets Clique em Extensões → Apps Script
2. Cole este código e salve:
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName("Lançamentos"); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.data, data.mes, data.natureza, data.direcao, data.tipo, data.categoria, data.descricao, data.conta, data.valor, data.status, data.observacao, "" ]); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
3. Implante como Web App Clique em Implantar → Nova implantação → Web App
Execute como: Eu | Acesso: Qualquer pessoa
